# AWS Architecture — Microservice Patterns & Design

A personal knowledge base documenting commonly used AWS services, architecture patterns,
and design principles for building microservice-based systems on AWS.

---

## 📌 What's Inside

This repository covers real-world patterns and best practices I've studied and applied,
organized by topic:

- **Compute** — ECS, EKS, Lambda, EC2 Auto Scaling
- **Networking** — VPC, API Gateway, Load Balancers, Service Mesh
- **Messaging** — SQS, SNS, EventBridge, Kafka (MSK)
- **Storage & Database** — RDS, DynamoDB, S3, ElastiCache
- **Observability** — CloudWatch, X-Ray, OpenTelemetry
- **Security** — IAM, Secrets Manager, WAF, KMS
- **CI/CD** — CodePipeline, CodeBuild, GitHub Actions on AWS
- **Infrastructure as Code** — CloudFormation, CDK, Terraform
